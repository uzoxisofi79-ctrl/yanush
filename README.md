
# ЯНУШ (JANUS) | AI Психосимулятор для Педагогов

Профессиональный тренажер на базе Google Gemini 3.0, имитирующий сложные разговоры с подростками на основе акцентуаций по А. Личко.

## 🚀 Коммерческая готовность

Приложение готово к использованию как MVP (Minimum Viable Product).

### Основные возможности:
- **Chaos Engine**: Динамическая сборка сценариев (Психотип + Инцидент + Фоновый конфликт).
- **Secure Auth**: Регистрация пользователей с SHA-256 хэшированием паролей.
- **Paywall**: Разграничение доступа к сложным акцентуациям (Premium).
- **AI Analysis**: Вердикт «комиссии экспертов» после каждого сеанса.

### Как запустить настоящую оплату:
1. В файле `constants.ts` укажите ваш `PAYMENT_PROVIDER_URL`.
2. В `SetupScreen.tsx` замените имитацию ожидания в функции `processPayment` на редирект: `window.location.href = ...`.
3. Для полноценной защиты данных рекомендуется перенести `localStorage` на **Supabase** (PostgreSQL) или **Firebase Auth**.

## 🛠 Установка

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/your-username/janus-ai.git
   ```
2. **Установите зависимости**:
   ```bash
   npm install
   ```
3. **Настройте API Key**:
   Создайте `.env.local` и добавьте `API_KEY=ваш_ключ_из_google_ai_studio`.

## 🛡 Безопасность
- Пароли хэшируются на клиенте.
- Внедрен `SecurityShield` для защиты от инспекции кода в браузере.
- Поддерживается Vercel Proxy для обхода блокировок API.
